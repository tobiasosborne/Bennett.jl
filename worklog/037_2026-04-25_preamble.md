**Rejected alternatives:**
- Topic-based grouping (`worklog/T5/`, `worklog/memory-plan/`) — sessions span
  multiple topics; grouping would fragment narratives.
- Date-only naming without `NNN_` prefix — disk sort wouldn't preserve source
  order, and the source has out-of-order dates within days.
- Splitting the 579-line monolith at paragraph boundaries — would harm
  readability of the catalogue→beads narrative. Kept as one unit.
- Extracting the embedded reference content (Project purpose / Repository

  superseded by CLAUDE.md's File Structure block (refreshed earlier today via

  now to preserve byte-for-byte history; future cleanup may extract.
